Adds music post type with taxonomy Genre and music tag.

== Description ==

Adds music post type with taxonomy Genre and music tag. Also adss custom meta box to save music meta information like Composer Name, Publisher, Year of recording, Additional Contributors, URL, Price.

= Help & Support =

Visit our Help Center if you have any questions, our friendly Support Team is happy to help - <https://khanzeeshan.in/>

== Installation ==

1. Upload the wp-music` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin via the 'Plugins' menu in WordPress.
3. Plugin settings are located under the Musics menu.
